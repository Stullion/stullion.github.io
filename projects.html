Stullio - an app for parents to record their own read-along books

Web Application

Growing up I had books with a companion cassette tape. The tape would read the book, then chime when it was time to turn the page. The only 21st century versions of that are prohibitively expensive. So this app lets you record a book yourself, indicate page turns, and then your kids can follow along with the book without you - no screen time required

Costco Receipt Tracker

Web Application

This was a silly project that was less meant to be used and more to test out implementing OCR via vibe-coded project. Many items I buy from Costco at regular intervals (like every other week), but I don't remember how long it's been for every item. This app takes pictures or pdfs of receipts and tracks what was purchased so I can see when the last time I bought an item. Two weeks since I last bought hummus? Better grab a tub! Heads up, this version stores my receipts in GitHub, so all the data is public.
